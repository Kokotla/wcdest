About WCD

WCD is a program for clustering expressed sequence tags. It is published under the GPL2. It has been tested on MacOSX, Linux and Windows and is parallelised for pthreads (multi-core) and MPI. Extensive documentation is available. The underlying algorithsm are described in papers listed below

Download

You can download WCD from GitHub. The latest available version can be found there. Binaries can be found at http://www.bioinf.wits.ac.za/software/wcd but these may not be the latest versions

Papers

If you use WCD, please cite

  • S Hazelhurst, Z Lipták. KABOOM! A new suffix array based algorithm for clustering expression data. Bioinformatics 27 (24), 3348-3355

Other papers are

  • S Hazelhurst, W Hide, Z Lipták, R Nogueira, R Starfield. 2008. An overview of the wcd EST clustering tool. Bioinformatics 24 (13), 1542-1546
  • S Hazelhurst 2008. Algorithms for clustering expressed sequence tags: the wcd tool. South African Computer Journal 40, 51-62
